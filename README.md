Sprocket
========

**Gameboy Emulator in TypeScript/HTML5**

## Intro
This is my first attempt at a Gameboy emulator. I'm well aware of the fact
that Javascript isn't an ideal language for writing a serious emulator... 
but I understand the HTML5 canvas API much better than C++ alternatives like 
SDL or DirectX. I'd also like to try integrating this emulator into some other 
web-based projects I have in mind.

*So here goes nothing!...*
