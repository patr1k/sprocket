Sprocket
========

**Gameboy Emulator in WebAssembly/HTML5**

## Intro
The goal of this project is to implement a GameBoy emulator as a WebAssembly binary,
allowing it to be used *efficiently* in the browser. The project is divided into two
logical parts, the **core** and the **lib**.
